This is a react application manage the user to improve the reading experience. As the application has three shelves. Currently reading, want to read, and read to manage the reading, the user is also able to search for any book and add it to any shelf.

#to run the project
1. npm install: to installs a package and any packages that the project depends on.
2. npm start: to Runs the app in the development mode. And open http://localhost:3000 to view it in your browser.
